# EmployeeSalaryUpdater

This is a C program that calculates salary increases for employees based on their gender and salary range. Originally written for Borland C, it has been updated to run on **Windows 11 using Visual Studio Code**.

## Features

- Input gender and salary for 45 employees.
- Calculate salary increases based on rules:
  - Female employees with salary < 350,000 receive 12% increase.
  - Male employees with salary between 350,000 and 676,000 receive 7% increase.
  - Other cases: salary remains unchanged.
- Display previous and updated salaries in a tabular format.
- Query individual employees to see their updated salary.
- Display the highest updated salary and its employee positions.

## Requirements

- Windows 11
- Visual Studio Code
- C compiler (e.g., GCC or MSVC)
